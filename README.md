# Food Delivery UI Website

This is a modern and responsive food delivery UI website made using **HTML, CSS, and JavaScript**.  
It includes sections like Hero, How It Works, Menu, Reviews, App Promotion, and Contact.

## Features
- Clean and attractive UI design  
- Fully responsive layout  
- Hero section with CTA  
- How it works cards  
- Dynamic menu area  
- Customer review slider  
- App promotion section  
- Newsletter/Subscribe section  

## Technologies Used
- HTML  
- CSS  
- JavaScript  
- Font Awesome  
- Swiper.js  

## How to Use
1. Download or clone the project  
2. Open `index.html` in your browser  
3. All images are inside the `images` folder
